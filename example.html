CSSGrid

CSSGrid is an easier-to-use grid. No .first, no .last, just nestable grid slots.

Purpose

CSSGrid is an effort to use less classes to achieve a comprehensive and flexible grid. Each element in the grid only requires one class. That means no .first or .last classes anywhere.

Nesting

Nesting is no problem. You can nest elements as deeply as you desire, and they'll just work. No special classes or elements required.

Weight

CSSGrid means less junk in your HTML in favor of a little bit of extra CSS. Even then, SimpleGrid weighs in at only 2.3kb with comments and readable styles.

How to Use It

Using CSSGrid is easy. It starts with a wrapper which is called .grid in this example (but you could change it to whatever you'd like). Once you've got the wrapper in place, create divs with a class of .row. Inside each row, create a set of columns for content areas.

Columns

CSSGrid supports both 4 and 6 column structures. Each column class is prefixed with a c4 or c6 designator to indicate which column type you are using. Then it's a simple matter of numbering the columns from 1-n.

Spanning columns is done by simply running numbers together, for example: 4col-12.
